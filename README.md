# 校园异能扭蛋（MVP）

一个可运行的手机文字抽卡游戏原型（命令行版），用于快速验证核心循环：

- 抽卡（单抽/十连）
- 养成（角色升级）
- 主线推进（自动战斗）
- 资源循环（钻石/金币/体力）

## 运行方式

```bash
python3 main.py
```

## 当前规则（MVP）

- 初始资源：1200钻石、1000金币、12体力
- 单抽消耗120钻石
- 十连消耗1080钻石，且至少1张SR
- 主线每次消耗1体力，胜利可得金币与钻石
- 升级角色消耗金币，提升战力
