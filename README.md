# 校园异能扭蛋（MVP+）

这是一个可运行的文字抽卡原型，增强了基础玩法的“可持续游玩性”。

## 运行方式

```bash
python3 main.py
```

## 新增玩法（相较初版）

- **校园事件系统**：每次事件有双选项，不同风险/收益。
- **手动编队系统**：可自选3名角色出战，不再只是“自动取战力前三”。
- **属性克制与阵容加成**：战斗中根据敌方属性触发胜率增益。
- **每日任务系统**：抽卡/推图/事件目标达成后领取奖励。

## 核心循环（当前版本）

1. 抽卡补强角色池
2. 编队并利用属性克制推进主线
3. 参与校园事件获取高风险高收益资源
4. 完成每日任务领取额外资源
5. 升级角色并继续推图

## 后续可迭代方向

- 本地 JSON 存档
- 新手引导和章节剧情文本
- 角色技能（主动/被动）
- 限时卡池与保底计数可视化
