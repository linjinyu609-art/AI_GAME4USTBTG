# 校园异能扭蛋（MVP++）

这是一个可运行的文字抽卡原型，已加入“章节怪物成长 + 属性规划建议”，解决后期推进缺少明确养成方向的问题。

## 运行方式

```bash
python3 main.py
```

## 当前重点增强

- **怪物成长模型**
  - 敌人等级：`chapter * 5 + stage * 2`
  - 第 5/10 关分别为精英/Boss（倍率提升）
  - 敌方属性：70% 按章节主属性出现，30% 随机
- **章节属性规划**
  - 新增菜单 `10. 章节属性规划`
  - 展示当前章/下一章主怪属性与推荐克制属性
  - 给出下一章节参考敌人战力，辅助决定是否先养成再推进

## 已有玩法

- 抽卡（单抽/十连保底 SR）
- 角色升级
- 手动编队
- 主线战斗（含属性克制和阵容加成）
- 校园事件（双选项风险收益）
- 每日任务奖励
