# 校园异能扭蛋（Playable CLI Build）

这是一个可运行的文字抽卡/养成/RPG 原型，当前版本重点更新了**游戏玩法层**（不只是数据库）。

## 运行

```bash
python3 main.py
```

## 本次玩法更新亮点

- **芯片工坊系统**（新）
  - 获取碎片后可合成芯片并装配到角色
  - 芯片有职业适配加成，影响战斗实际战力
- **队伍协同系统**（新）
  - 3职业互补、3属性多样会触发队伍协同加成
- **深渊试炼模式**（新）
  - 独立的3层挑战玩法，使用试炼票进入
  - 按试炼分数结算金币/钻石/碎片
- **失败反馈增强**（新）
  - 会提示克制缺失、等级不足、芯片缺失等原因

## 主要玩法

- 抽卡：单抽、十连、保底机制
- 阵容：手动编队、技能触发、元素克制
- 推图：章节/关卡推进、精英/Boss敌人
- 成长：角色升级、升星（重复卡转化）
- 事件：1200+校园事件池（风险收益）
- 任务：每日任务、章节任务板
- 账号：账号等级、存档读写、体力循环

## 代码结构

- `main.py`：入口
- `campus_game/engine.py`：游戏核心系统（玩法逻辑）
- `campus_game/content_database.py`：大规模内容库（角色/敌人/事件/章节/技能）
- `PRODUCT_ROADMAP.md`：后续版本路线图
