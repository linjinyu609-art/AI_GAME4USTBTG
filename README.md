# 校园异能扭蛋（Playable CLI Build）

这是一个可运行的文字抽卡/养成/RPG 原型，已升级为多系统可玩版本：

- 抽卡：单抽、十连、保底机制
- 阵容：手动编队、技能触发、元素克制
- 推图：章节/关卡推进、精英/Boss敌人
- 成长：角色升级、升星（重复卡转化）
- 事件：1200+校园事件池（风险收益）
- 任务：每日任务、章节任务板
- 账号：账号等级、存档读写、体力循环

## 运行

```bash
python3 main.py
```

## 代码结构

- `main.py`：入口
- `campus_game/engine.py`：游戏核心系统
- `campus_game/content_database.py`：大规模内容库（角色/敌人/事件/章节/技能）
- `PRODUCT_ROADMAP.md`：后续版本路线图

## 快速自测建议

1. 十连抽卡，查看不同稀有度与技能
2. 编队后推进主线，观察技能倍率和战斗日志
3. 执行校园事件，体验风险收益
4. 完成每日任务并领取奖励
5. 退出后重进，验证存档
